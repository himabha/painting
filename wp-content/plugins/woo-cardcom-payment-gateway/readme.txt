== Installation ==

= Minimum Requirements =

* WordPress 3.9 or later
* PHP version 5.6 or later
* MySQL version 5.6 or later

* WooCommerce 2.7 or later

1. Upload `woocommerce-cardcom-gateway` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
== Now you need to configure cardcom plugin ==
3. You need to get terminal number and api username, you supposed to get to email when register to Cardcom.
In case you don't have this details:
log in to cardcom system in navbar choose settings->1.company and user settings->6.manage api keys
4. In WordPress admin panel go to Woocammerce settings->chakout->cardcom
5. Eanble the plugin
6. Regarding filling the rest of field please see cardcom kb

== Frequently asked questions ==

= A question that someone might have =

Q: Where to get Terminal number
A: If you didn't get an email on register to cardcom just log to https://secure.cardcom.co.il on top of the page there it is.
Q: Where to get api username
A: og in to cardcom system in navbar choose settings->1.company and user settings->6.manage api keys
Q: What is my user name and password to cardcom system
A: If you didn't get an email on register to cardcom you can contact with Cardcom support http://www.cardcom.co.il/Pages2/ContactUs.aspx
